# Wordle-Project

<h2>Description</h2>
Recreated the classic game "Wordle" through Java, leveraging core object-oriented programming concepts such as encapsulation, inheritance, and polymorphism. This project encompasses the creation of a user-friendly interface, incorporating features like randomized word selection and dynamic score tracking to amplify player engagement. 
<br />


<h2>Languages and Utilities Used</h2>

- <b>Java</b> 
- <b>Object Oriented Prorgaming</b>
- <b>VsCode<b>
